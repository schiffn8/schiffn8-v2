## Challenge

The studio had outgrown its original identity — built quickly for launch, it no longer reflected the quality of work being produced. The brief was to create a system that would hold up across print, digital, and environmental applications for the next five years.

The core tension: the studio wanted to feel premium without being cold, and accessible without being casual.

## Approach

The identity anchors on a single wordmark — set in a cut of Freight Display, tracked tight, with custom spacing on the terminal characters. No logomark. The reasoning: studios that rely on symbols are betting that a client will remember the symbol; studios that rely on the name are betting that the name becomes the symbol over time.

Color system: near-black on warm white for day-to-day materials, with a single signal color (a specific orange — Pantone 021 C) reserved for moments that need to cut through. No gradients, no textures.

The type system pairs the display cut with a neutral sans (Inter) for body copy and a monospace (JetBrains Mono) for metadata and labels — a three-register system that allows any document to establish hierarchy without relying on weight variation.

## Outcome

The system launched across the studio's website, print collateral, and signage. The website went live in Q3 2025.
